# Projekt: klasyfikacja rezerwacji hotelowych

Celem projektu będzie stworzenie modelu klasyfikacyjnego, który będzie przewidywał, czy dana rezerwacja pokoju w hotelu zostanie anulowana.

Dane: [Hotel Reservations Dataset](https://www.kaggle.com/datasets/ahsan81/hotel-reservations-classification-dataset)

Zespoły: 3-osobowe, zostaną wylosowane

---

## Etapy projektu

### 1. Eksploracyjna analiza danych

- Wczytanie danych, rozmiar, typy zmiennych, statystyki opisowe
- Zrozumienie zmiennych
- Rozkłady cech, korelacje między nimi, wykresy pokazujące zależności
- Czy występują braki, błędy, wartości odstające?

### 2. Przetwarzanie danych

- Podział na zbiór treningowy, walidacyjny, testowy
- Czyszczenie danych z braków, błędów, duplikatów
- Czy robimy coś z outlierami?
- Encoding zmiennych kategorycznych
- Skalowanie, transformacje

### 3. Trening i ewaluacja modeli

- Wybór modeli do przetestowania
- Trening modeli, tuning hiperparametrów
- Ocena jakosci modeli, wybór metryk
- Wybór najlepszego modelu

### 4. Interpretacja modelu

- Istotność zmiennych
- Wnioski biznesowe

### 5. Podsumowanie

- Prezentacja, dokumentacja podsumowujące projekt

## Oczekiwane efekty
